# 🎓 Student Management System (Dynamic OOP in C++)

## 📌 Description
This project is a Student Management System developed using Object-Oriented Programming (OOP) concepts in C++. It uses dynamic memory allocation to store and manage student records efficiently.

---

## 🎯 Objective
- Implement OOP concepts in C++  
- Use dynamic memory allocation  
- Manage student data effectively  

---

## 🛠️ Tech Stack
- C++  
- OOP Concepts  
- Dynamic Memory Allocation  

---

## 🧠 Concepts Used
- Classes and Objects  
- Constructors  
- Pointers  
- Dynamic Memory (new/delete)  

---

## ⚙️ Features
- Add student details  
- Display student records  
- Dynamic memory handling  
- Clean and simple interface  

---

## 🔍 Working
1. User enters number of students  
2. Program dynamically allocates memory  
3. User inputs student details  
4. Data is stored using objects  
5. Records are displayed  

---

## ▶️ How to Run

1. Compile:
   g++ student_management.cpp  

2. Run:
   ./a.out
